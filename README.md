# Hotel Ordering Application

## 📌 Project Overview

The Hotel Ordering Application is a full-stack web application designed to simplify food ordering in restaurants and hotels. The system enables customers to browse menus, place orders, manage their profiles, make secure payments using a wallet, and provide feedback after completing an order.

The project was developed following the Agile Scrum methodology with multiple sprints to ensure iterative development, continuous testing, and efficient delivery of features.

---

## 🎯 Objectives

- Develop a user-friendly hotel food ordering system.
- Simplify online food ordering and payment.
- Provide secure user authentication and profile management.
- Enable wallet-based payments and order tracking.
- Improve customer experience through feedback collection.
